---
title: MySQL
description: "Use MySQL tables as sources and sinks"
---

Support for using MySQL CDC as a source, and MySQL tables as a sink is supported via Debezium.
See the [Postgres](/connectors/postgres) docs for more information on how to configure the connector.
More complete documentation for MySQL is coming, and in the meantime please reach out to us
at support@arroyo.systems or on [discord](https://discord.gg/cjCr5rVmyR) with any questions.
