---
title: Scalar Functions
description: 'Supported Scalar Functions in Arroyo SQL'
---

Arroyo's Scalar function support mainly mirrors that of DataFusion, 
which in turn uses the semantics of the equivalent Postgres functions.

### Math Functions

<Accordion title="abs">
Returns the absolute value of a number.
```sql
abs(numeric_expression)
```
</Accordion>


<Accordion title="acos">
Returns the arc cosine of a number.
```sql
acos(numeric_expression)
```
</Accordion>

<Accordion title="acosh">
Returns the inverse hyperbolic cosine of a number.
```sql
acosh(numeric_expression)
```
</Accordion>

<Accordion title="asin">
Returns the arc sine of a number.
```sql
asin(numeric_expression)
```
</Accordion>

<Accordion title="asinh">
Returns the inverse hyperbolic sine of a number.
```sql
asinh(numeric_expression)
```
</Accordion>

<Accordion title="atan">
Returns the arc tangent of a number.
```sql
atan(numeric_expression)
```
</Accordion>

<Accordion title="ceil">
Returns the smallest integer greater than, or equal to, a number.
```sql
ceil(numeric_expression)
```
</Accordion>

<Accordion title="cos">
Returns the cosine of a number.
```sql
cos(numeric_expression)
```
</Accordion>

<Accordion title="cosh">
Returns the hyperbolic cosine of a number.
```sql
cosh(numeric_expression)
```
</Accordion>

<Accordion title="floor">  
Returns the largest integer less than, or equal to, a number.
```sql
floor(numeric_expression)
```
</Accordion>

<Accordion title="exp">
Returns e raised to the power of a number.
```sql
exp(numeric_expression)
```
</Accordion>

<Accordion title="ln">
Returns the natural logarithm of a number.
```sql
ln(numeric_expression)
```
</Accordion>

<Accordion title="log2">
Returns the base 2 logarithm of a number.
```sql
log2(numeric_expression)
```
</Accordion>

<Accordion title="signum">
Returns the sign of a number.
```sql
signum(numeric_expression)
```
</Accordion>

<Accordion title="sin">
Returns the sine of a number.
```sql
sin(numeric_expression)
```
</Accordion>

<Accordion title="sinh">
Returns the hyperbolic sine of a number.
```sql
sinh(numeric_expression)
```
</Accordion>
